# 한글 간단 편집기 (HWPX)

브라우저에서 `.hwpx` 파일을 열고 **문구만 간단히 수정**해서 다시 받는 무료 편집기.

👉 **[바로 사용하기](https://hyshin6664.github.io/hwpx-editor/)**

## 특징

- **서버 없음**: 파일이 사용자 브라우저 안에서만 처리됩니다. 어디에도 업로드되지 않습니다.
- **설치 없음**: 어떤 브라우저에서도 됩니다 — 맥 · 윈도우 · 안드로이드 · 아이폰.
- **광고 · 추적 없음**: 정적 HTML 한 페이지가 전부입니다.
- **오픈소스**: MIT 라이선스.

## 이런 작업에 쓰세요

- 이름 · 날짜 · 금액 같은 문구만 살짝 바꾸기
- 오타 수정
- 같은 양식의 문서를 사람마다 조금씩 다르게 만들기

## 한계

- 표 · 이미지 · 서식 편집은 지원하지 않습니다 (텍스트 문구만).
- 구형 `.hwp` 파일은 지원하지 않습니다 — 한컴오피스에서 *다른 이름으로 저장 → HWPX(\*.hwpx)* 로 변환 후 사용해주세요.

## 기술 스택

- Vanilla JavaScript
- [JSZip](https://stuk.github.io/jszip/) (CDN)
- HTML 한 파일 (`index.html`)
- GitHub Pages 호스팅

## 동작 원리

HWPX는 OWPML 규격의 ZIP + XML 컨테이너입니다. 이 편집기는:

1. ZIP 컨테이너를 풀어 `Contents/section*.xml` 들을 읽습니다.
2. 각 XML 안의 `<hp:t>` 텍스트 노드만 화면에 표시합니다.
3. 사용자가 텍스트만 수정하면 `<hp:t>` 의 textContent 만 갱신합니다 (id, 참조, 서식 모두 보존).
4. ZIP을 다시 묶어 다운로드합니다.

문단 추가/삭제 · 표 구조 변경 · 서식 변경은 일체 하지 않으므로 한컴오피스에서 안전하게 열립니다.
